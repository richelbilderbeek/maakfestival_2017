# maakfestival 2017

Activiteiten van De Jonge Onderzoekers, DIYbio en Hive5 op het [Maakfestival](https://www.maakfestival.nl)

## Agenda

 * Maandag 1 mei 
   * 12:00: deadline inschrijven als maker
 * Vrijdag 19 mei 2017
   * 18:00-22:00: Opbouw
 * Zaterdag 20 mei 2017: opbouw en festival
   * 8:00-10:00: Opbouw
   * 10:00-17:00: Publiek

Afkorting|Aanduiding
